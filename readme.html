About S3TBX

Introduction

The S3TBX software is a set of executable tools and an application programming interface (API) facilitating the use of the data products of a number of earth observation satellites. Its main purpose is to support the data of the OLCI and SLSTR sensors onboard the ESA Sentinel-3 platform. S3TBX is developed under contract to ESA/ESRIN.

Directory Structure

This is the structure of the S3TBX installation directory:

  • bin - S3TBX executable and S3TBX command-line tools.
  • config - S3TBX configuration file.
  • lib - Common libraries used by S3TBX.
  • modules - Installed S3TBX modules and plugins.
  • licenses - 3rd party licenses and EULAs of libraries.

If S3TBX is started the first time, a .s3tbx directory is created in the user's home directory. S3TBX uses this directory to store user preferences, auxiliary and temporary data.

Third Party Content

S3TBX includes some items that have been sourced from third parties. The licenses for these items are provided in the licenses directory (see above).
